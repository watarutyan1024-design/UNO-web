UNO Classic Web Game

ブラウザで動作する、シンプルで直感的なUNOゲーム。
1人のプレイヤーと3人のCPUによる4人対戦モード。

「スタート」ボタンを押してゲーム開始。

🎮 操作ガイド

カードを出す: 自分の番で、光っている（少し浮いている）カードをクリック。

カードを引く: 出せるカードがない場合、左側の黒い山札をクリック。

色を変える: ワイルドカード（黒）を出した際、中央に出る4色パネルから1つ選ぶ。

📝 実装済みルール

基本カード: 0-9の数字カード（4色）。

アクションカード:

Skip (Ø): 次の人の番を飛ばす。

Reverse (⇄): 回り順を逆にする。

+2: 次の人に2枚引かせて休み。

ワイルドカード:

Wild (W): 場の色を自由に変える。

+4: 色を変えつつ、次の人に4枚引かせて休み。

勝利判定: 手札が0枚になった順にランキングを表示。

🛠 技術構成

Frontend: HTML5, JavaScript (ES6+).

Styling: Tailwind CSS (CDN).

Fonts: Google Fonts (Roboto 900).

📂 ファイル構成

uno_game.html: 全ロジック、スタイル、マークアップを統合したメインファイル。

Enjoy the game!
